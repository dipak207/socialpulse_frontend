# SocialPulse Intelligence - Frontend

Next.js 14 App Router frontend with Tailwind CSS and shadcn/ui.

## Setup
1. `npm install`
2. Copy `.env.local.example` to `.env.local`
3. `npm run dev`

## Features
- Glassmorphism UI
- Real-time job polling
- Interactive charts with Recharts
- Framer Motion animations
